## 0.1.6

### Patch Changes

- 7aa81e7: - @toolbox-ts/colors Initial Release
  - @toolbox-ts/utils Num module rework

      - UnitInterval module utilities for [0,1] range
      - Bits module for working with Bit data
      - round function with decimal point
